## To preview the frontend:  
`cd frontend`  
`npm install` (you only do this once)  
`npm run dev`

## To preview the backend:  
`cd backend`  
### Windows:
`.\mvnw spring-boot:run`
### MacOS/Linux:
`./mvnw spring-boot:run`
